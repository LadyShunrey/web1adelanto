El práctico Parte 1

1) ¿Cómo es la estructura de un documento HTML? Investigar para qué sirve cada sección del documento. Un documento HTML siempre tiene un "!DOCTYPE, un html, un head y un body". El DOCTYPE le dice al navegador que estamos haciendo un html, y que es html 5. El tag html viene siempre después del DOCTYPE. Dentro del html va a estar todo mi archivo html y contenido del sitio y está dividido en dos partes: head y body. El head va todo lo que no ve el usuario, es info para el navegador el usuario no lo ve (meta y title). En el body va todo lo que ve el usuario, todo el contenido de la página.

2) ¿Para qué sirven los estilos web? ¿Cómo se aplican los estilos al contenido de una página web? Los estilos sirven para darle forma y color a la página. Los estilos se aplican al contenido mediante un link en el html al css con una hoja de estilos externa.

3) ¿Qué tipos de selectores existen en CSS? ¿Cuándo conviene usar cada uno de ellos? Existen 3 tipos de selectores, Tipo, Clase e ID. El de Tipo se usa para todos los parrafos en general. El de Clase se usa para darle un atributo especial a un parrafo. El de ID se usa para creara un estilo único y no es recomendado.
